# IEEE Ignite 2026 – Event Website

A modern, responsive, and interactive event website built for **IEEE Ignite 2026**.  
Designed with a clean UI, smooth animations, and dynamic features using **HTML, CSS, and JavaScript**.

---

##  Overview

IEEE Ignite 2026 is a flagship technical event organized by the IEEE Student Branch.  
This website serves as the official event landing page, showcasing event details, schedules, FAQs, and interactive features for participants.

---

##  Features

- Responsive modern UI (desktop & mobile)
- Smooth scrolling navigation with active section highlighting
- Event filtering by category (Technical / Professional / Interactive)
- Real-time event search with clear option
- Day-wise schedule timeline switching
- Interactive FAQ accordion
- Mobile-friendly hamburger menu
- Dark futuristic theme with glowing accents

##  File Details

### index.html
- Contains the complete page structure
- Includes sections:
  - Home
  - About IEEE Ignite
  - Events with filters & search
  - Schedule (Day 1 & Day 2)
  - FAQ
  - Footer
- Uses Google Fonts 

---

### style.css
- Implements a dark, modern UI theme
- Uses CSS variables for easy theme management
- Fully responsive layout
- Smooth hover effects
- Card-based UI design
- Timeline and FAQ animations

---

### script.js
Handles all dynamic behavior:
- Navbar scroll effect
- Mobile menu toggle
- Active nav link highlighting
- Event filtering & searching
- FAQ accordion logic
- Schedule tab switching
- No-results handling for search
